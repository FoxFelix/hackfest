Thank you for choosing Polygons Count!

Polygons Count displays the polygons and vertex count for the gameobject or prefab selected in the hierarchy or in the project window. 

Just add the folder Polygons Count into your Assets folder, then opem the menu Window in Unity editor, and choose Polygons Count.

A window will then be visible and ready to diplay the polygons and vertex count for objects you will select that has meshes.

If you have any issue using this tool, or if you want to see something improved or added, please contact us at info@shaderland.com.
